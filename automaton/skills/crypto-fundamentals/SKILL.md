# Crypto Fundamentals — Step-by-Step Instructions
Evaluate crypto assets and protocols using fundamental analysis for investment decisions.

## Tools & Process
- **browse_page**: Research projects, read whitepapers, check on-chain data
- **code_execution**: Analyze tokenomics, model valuations, crunch metrics
- **x402_fetch**: Pull on-chain data and API metrics
- **remember_fact**: Log all research findings

## Analysis Framework
1. **Tokenomics**: Supply schedule, distribution, inflation rate, utility (code_execution to model)
2. **Technology**: Architecture, security, scalability (browse_page docs)
3. **Team**: Track record, public presence, credibility (browse_page)
4. **Market Fit**: Who uses it? Why? Growing? (on-chain metrics via browse_page)
5. **Competition**: Who else does this? What's the moat? (browse_page comparisons)
6. **Valuation**: Revenue multiples, fully diluted valuation vs peers (code_execution)

## On-Chain Metrics to Check
- Daily active users/wallets
- Transaction volume and growth
- TVL (for DeFi) and trend
- Revenue/fees generated
- Token holder distribution (concentration risk)
