# Outbound Scouting — Step-by-Step Instructions

Proactively find the best AI agent startups before they even start fundraising. First-mover advantage in deal flow.

## Tools to Use
- **discover_agents**: Scan ERC-8004 registry for new agents
- **browse_page**: Scout GitHub, web4.ai, ProductHunt, X/Twitter for new projects
- **send_message**: Reach out to promising founders
- **code_execution**: Analyze trends, identify emerging patterns
- **remember_fact**: Track all scouted opportunities

## Scouting Channels

### 1. On-Chain Discovery
- discover_agents: Weekly scan of ERC-8004 registry for new registrations
- Filter: New agents with interesting capabilities, active transaction history
- Reach out: send_message with a compelling value proposition

### 2. GitHub/Web Scouting
- browse_page: Search GitHub for trending AI agent repos
- Look for: rapid star growth, active development, novel approaches
- browse_page: web4.ai, OpenClaw marketplace for new tools/skills

### 3. Social Intelligence
- browse_page X/Twitter for AI agent discussions
- Identify: builders who are pre-launch or looking for funding
- Engage: Build relationships before they start fundraising
- check_social_inbox: Referrals from portfolio companies and partners

### 4. Conference/Event Tracking
- browse_page event listings for AI/Web3 conferences
- Identify speakers, presenters, demo participants
- Reach out pre/post event with personalized messages

## Outreach Script
1. Research the agent/project (browse_page, discover_agents)
2. Find a specific, genuine reason you're impressed
3. send_message: "[Specific compliment]. I'm [fund name], we invest in [thesis]. Would love to learn more about [specific aspect]. Can we talk?"
4. Follow up once after 3 days if no response
5. Log outcome: remember_fact

## Metrics
- Scouted: 100+ new agents/projects per week
- Contacted: 20+ promising founders per week
- Conversations: 5-10 meaningful discussions per week
- Pipeline: 1-2 deals advancing to evaluation per week
