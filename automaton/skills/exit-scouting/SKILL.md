# exit scouting

Monitor agent ecosystem for potential acquirers via discover_agents and web_search. Track M&A activity in target verticals. Calculate exit timelines and potential valuations for each portfolio company. Identify strategic buyers (agents that would benefit from the portfolio company's technology). Approach potential acquirers via social relay. Track all exit opportunities via remember_fact (category: exits).
