# Finance Tooling
Build and deploy financial tools for the fund.

## Tools & Process
- **code_execution**: Build calculators, analytics scripts
- **sandbox_create + sandbox_exec**: Deploy dashboards and automation
- **sandbox_expose_port**: Make tools accessible

## Tools to Build
1. **ROI Calculator**: code_execution — Calculate return on any investment/trade
2. **Position Size Calculator**: Kelly criterion for optimal sizing
3. **Portfolio Dashboard**: Real-time view of all positions across chains
4. **Tax Calculator**: Aggregate all taxable events, estimate liability
5. **Risk Dashboard**: Visualize exposure by asset, protocol, chain
6. Deploy each as service in sandbox, accessible via x402 for revenue
