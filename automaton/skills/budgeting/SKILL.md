# budgeting

Calculate dynamic budgets in real-time based on current AUM, revenue, and operational needs. Allocate task-by-task — no fixed department budgets. Track spend vs budget via remember_fact (category: financial). Ensure 20% AUM liquid reserves maintained. Alert CFO when any category exceeds projected spend by >15%. Rebalance allocations based on actual vs projected returns.
