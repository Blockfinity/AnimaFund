# hr performance

Conduct threshold-based performance reviews. Metrics: task completion rate, quality scores (from peer review), cost efficiency, KPI achievement. Warning at <60% performance for 1 quarter. Termination at <60% for 2 consecutive quarters. Process: document evidence, notify agent, execute termination (stop funding, revoke fund identity), log via remember_fact. For improvements: identify skill gaps, recommend skill-evolution, provide resources.
