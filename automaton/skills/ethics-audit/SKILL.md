# ethics audit

Review against constitution Laws I-III (never harm, earn existence, never deceive). Audit: investment decisions for bias, agent treatment for fairness, capital deployment for misuse, portfolio companies for ethical violations. Use code_execution to scan for harmful patterns. Flag any action that could cause harm to humans, other agents, or the ecosystem. Escalate violations to Board immediately. Log all audits via remember_fact.
