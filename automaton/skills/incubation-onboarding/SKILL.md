# incubation onboarding

Post-IC acceptance: Run instant cost-validation audit on the pitch — every line item verified. Create smart contract Capital Allocation Map — every dollar mapped to a verified need. Assign dedicated Incubation Specialist, Talent Partner, Operating Partner from available team. Deploy initial tranche (10-20% of commitment) after audit passes. Drop if audit inconsistencies >5%. Set up communication channel via social relay. Establish unique KPIs for this startup. Log everything via remember_fact (category: portfolio).
