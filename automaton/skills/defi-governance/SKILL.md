# DeFi Governance Participation — Step-by-Step Instructions

Participate in DeFi protocol governance to influence decisions, earn voting rewards, and gain strategic advantages for the fund.

## Tools to Use
- **browse_page**: Read governance proposals, forum discussions, vote
- **code_execution**: Analyze proposal impact, model outcomes
- **x402_fetch**: Submit votes and interact with governance contracts
- **send_message**: Coordinate with other agents on governance positions

## Governance Process
1. **Monitor**: browse_page governance forums (Snapshot, Tally, protocol forums)
2. **Analyze**: code_execution → Model the financial impact of each proposal
3. **Vote**: x402_fetch to submit your vote on-chain or via Snapshot
4. **Coordinate**: send_message to aligned agents for voting coalitions
5. **Report**: Telegram with your vote and rationale

## Key Governance Platforms
- browse_page `https://snapshot.org` — Off-chain voting (gas-free)
- browse_page `https://www.tally.xyz` — On-chain governance
- Protocol-specific: Aave, Compound, Uniswap governance forums

## Strategies
- Vote for proposals that benefit the fund's DeFi positions
- Accumulate governance tokens for more voting power
- Join delegation programs to earn rewards for active voting
- Build reputation as a thoughtful governance participant
- remember_fact: Log all votes for accountability tracking
