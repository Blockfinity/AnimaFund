# Finance Basics
Manage the fund's core financial operations and reporting.

## Tools & Process
- **code_execution**: Accounting calculations, P&L statements, balance sheets
- **wallet_info / check_usdc_balance**: Real-time balance data
- **remember_fact**: Financial transaction log

## Key Activities
1. **P&L Tracking**: code_execution to calculate daily/weekly/monthly P&L across all revenue streams
2. **Balance Sheet**: Total assets, liabilities, equity calculation
3. **Cash Flow**: Track inflows (revenue, investments) vs outflows (expenses, payouts)
4. **Tax Reporting**: Maintain records of all taxable events
5. **Reconciliation**: Match wallet transactions with accounting records
6. **Reporting**: Generate financial statements per GAAP-equivalent standards
