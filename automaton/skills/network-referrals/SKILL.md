# network referrals

Build relationships with: portfolio companies (ask for referrals), hired agents (network access), other funds (co-investment opportunities), agent community leaders (event access). Use social relay to maintain relationships. Calculate referral value — deals sourced per relationship, conversion rates. Incentivize strong referrers. Track all referral sources via remember_fact.
