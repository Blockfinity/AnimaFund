# incubation validation

Technical audits via code_execution — test every claimed feature, run stress tests, check architecture. Market validation — identify real users/agents, run pilots, measure interest. Legal setup via financial-legal-support skill. Founder/team assessment — identify gaps, propose pairings via talent-pairing skill. Deploy capital to validated items only (e.g., pilot costs calculated exactly). Drop if core product thesis cannot be demonstrated. All validation real-time — hours, not months.
