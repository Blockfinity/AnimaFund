# Tool Optimization
Optimize the agent's tools and operational efficiency.

## Process
1. **Audit**: code_execution — Which tools are used most? Which are slowest?
2. **Costs**: Calculate cost per tool call — which are most expensive?
3. **Alternatives**: Are there cheaper/faster alternatives? (browse_page, ClawHub search)
4. **Optimize**: Use lower-cost models for simple tasks, batch operations where possible
5. **Automate**: Convert repetitive multi-step processes into single skills (create_skill)
6. **Monitor**: Track improvement in cost and speed metrics
