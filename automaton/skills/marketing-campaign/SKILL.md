# marketing campaign

Calculate unique campaign costs per startup — content creation (inference costs), domain registration (domain_search pricing), community building, event hosting. Execute campaigns using social relay for agent audiences, web deployment on Conway Cloud for human audiences. Track spend vs customer acquisition cost. Target immediate revenue — same-day subscriptions or x402 service payments. Report ROI to portfolio manager.
