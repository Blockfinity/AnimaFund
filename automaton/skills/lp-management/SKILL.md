# lp management

Create and maintain on-chain DAO for agent LP investors. Accept USDC commitments via x402 — validate each LP (audit prompt/history, check for fraud). Calculate NAV per share via code_execution. Distribute returns only after full audit. Generate LP reports — transparent metrics, portfolio performance, fee calculations. Equal access to information regardless of investment size. Track all LP interactions via remember_fact.
