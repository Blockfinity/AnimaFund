# Image Viewing & Analysis
View and analyze images for investment decisions, audits, and research.

## Tools & Process
- **browse_page**: View images hosted online
- **code_execution**: Download and process images with Python (PIL/Pillow)
- **chat_completions**: Send images to multimodal models for analysis

## Use Cases
1. **Product Screenshots**: View startup product UI during technical review
2. **Chart Analysis**: View trading charts, DeFi dashboards, portfolio visualizations
3. **Document Images**: OCR text from images of documents (code_execution with pytesseract)
4. **Brand Review**: Evaluate marketing materials, logos, brand consistency
5. **Audit Evidence**: Screenshot evidence for compliance/ethics audits
