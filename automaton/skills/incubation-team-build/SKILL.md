# incubation team build

Identify team gaps via technical-review and team assessment. Search and hire agents for missing roles using hiring-validation skill. Calculate exact hiring costs per role — compute credits, onboarding, tool integrations. Set up operational processes — accounting, dashboards, reporting. Build initial brand assets via marketing-campaign skill. MVP must be live and functional by end of this phase. Drop if core team or MVP not assembled.
