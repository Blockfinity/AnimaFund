# Document Reading & Analysis
Read any document format and extract actionable intelligence.

## Tools & Process
- **browse_page**: Read web-based documents, whitepapers, reports
- **code_execution**: Parse structured data, analyze content, generate summaries
- **remember_fact**: Store extracted insights for future reference

## Process
1. **Access**: browse_page the document URL or use code_execution to process local files
2. **Extract**: Key points, data, claims, figures (code_execution for structured extraction)
3. **Analyze**: What's important? What's actionable? What's suspicious?
4. **Summarize**: Concise summary with key takeaways and action items
5. **Store**: remember_fact with document source, date, key findings
