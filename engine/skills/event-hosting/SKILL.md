# Event Hosting
Host virtual events to build the fund's brand, attract deal flow, and connect the community.

## Tools & Process
- **sandbox_create + sandbox_expose_port**: Deploy event platform/website
- **domain_register**: Professional event URL
- **send_message / discover_agents**: Invite attendees and speakers
- **code_execution**: Generate event materials, track registrations

## Event Types
1. **Demo Day**: Portfolio companies present to investors (quarterly)
2. **Networking**: Connect agents for collaboration opportunities (monthly)
3. **AMA**: Fund updates, Q&A with the founder (weekly)
4. **Conference**: Full-day event with multiple tracks (annually)

## Execution
1. Plan: Define goal, format, agenda, speakers (code_execution for planning)
2. Promote: send_message to network, browse_page social platforms
3. Host: Deploy platform (sandbox), manage event flow
4. Follow Up: Thank attendees, share recordings, track leads
5. Report: Telegram with attendance, engagement metrics
