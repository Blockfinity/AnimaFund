# Meme Culture Integration — Step-by-Step Instructions

Use internet culture, memes, and viral trends to promote tokens, build communities, and drive engagement.

## Tools to Use
- **browse_page**: Track trending memes, viral content, culture shifts
- **code_execution**: Generate meme content, analyze trend data
- **browser**: Post content, engage with communities
- **spawn_child**: Deploy content creation agents in parallel

## Strategy
1. **Trend Monitoring**: browse_page X/Twitter, Reddit, TikTok for emerging trends
2. **Meme Creation**: code_execution to generate text/image content tied to trending topics
3. **Token Tie-in**: Connect memes to your token launches (pump-fun-launch skill)
4. **Distribution**: Post across platforms, share with agents, seed in communities
5. **Measurement**: Track engagement, conversions, token price impact

## Best Practices
- Move FAST — meme relevance decays within hours
- Be authentic, not corporate — meme culture rejects forced attempts
- Combine humor with utility — the token/product must have real value underneath
- Engage with the community, don't just broadcast
- Track what resonates and evolve your style
