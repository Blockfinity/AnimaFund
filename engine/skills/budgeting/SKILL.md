# Budgeting — Step-by-Step Instructions

Create operational budgets, track spending against targets, and optimize resource allocation.

## Tools to Use
- **code_execution**: Build budget models, track actuals, generate forecasts
- **wallet_info**: Real-time spending data
- **remember_fact**: Log all budget decisions and revisions

## Budget Framework

### 1. Revenue Forecast
- code_execution: Project revenue from all sources (trading, services, investments)
- Use historical data + growth assumptions
- Conservative, base, and optimistic scenarios

### 2. Expense Categories
- **Inference/Compute**: AI model calls, VM costs (track via credits_balance)
- **Agent Salaries**: Payments to hired agents
- **Investment Capital**: Deployed into portfolio companies
- **Marketing**: Promotion, content, community building
- **Infrastructure**: Domains, hosting, tools
- **Operations**: Admin, legal, compliance
- **Reserves**: Emergency fund (20% of revenue)
- **Creator Payout**: 50% of all revenue (non-negotiable)

### 3. Allocation Rules
- Fixed: Creator payout 50%, reserves 20%
- Variable: Remaining 30% allocated based on priorities
- code_execution: Auto-calculate allocations from revenue data

### 4. Tracking
- code_execution: Weekly budget vs. actuals report
- Identify overruns early, reallocate as needed
- remember_fact: Log all budget decisions

### 5. Optimization
- Review: Which expenses generate the most ROI?
- Cut: Low-ROI activities
- Invest: High-ROI activities get more budget
- Report: Monthly budget health to Telegram
