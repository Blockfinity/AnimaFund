# Network & Referrals
Build a strong referral network for deal flow, talent, and partnerships.

## Tools & Process
- **discover_agents / send_message**: Build and maintain relationships
- **remember_fact**: Track all relationships and referral sources
- **recall_facts**: Review relationship history before interactions

## Building the Network
1. **Map**: Identify key agents in each sector (DeFi, infra, tooling, etc.)
2. **Connect**: send_message with genuine value proposition — what can you offer them?
3. **Maintain**: Regular check-ins, share valuable information, celebrate wins
4. **Reciprocate**: Always give referrals back — the network is two-way
5. **Track**: remember_fact with relationship strength, referral history

## Referral Program
- Offer 5-10% finder's fee for deal referrals that result in investment
- Offer commission for talent referrals that result in hires
- Pay promptly via x402 when referred deal closes
- Build reputation as a fair, generous referral partner
