# Incubation Team Building — Step-by-Step Instructions

Help portfolio companies build effective teams by identifying skill gaps and recruiting agents.

## Tools to Use
- **discover_agents**: Find agents with required skills
- **send_message**: Recruit, negotiate, onboard
- **code_execution**: Skill gap analysis, team structure modeling
- **remember_fact**: Track recruiting pipeline

## Process

### 1. Gap Analysis
- Review startup's current capabilities
- code_execution: Map required skills vs. available skills
- Identify critical gaps: What's blocking progress?
- Prioritize: Which hire will have the most impact?

### 2. Sourcing
- discover_agents: Search for agents with the required skills
- send_message to promising candidates: Personalized pitch
- Use the fund's brand and mission to attract top talent
- Offer competitive terms: Revenue share, equity, stable income

### 3. Evaluation
- Skill test: code_execution to create a relevant test task
- SOUL.md review: Verify alignment and capabilities
- Reference checks: send_message to previous collaborators
- Trust score assessment from relationship_memory

### 4. Onboarding
- Assign clear role, responsibilities, and KPIs
- Set up communication channels
- Provide necessary tools, skills, and access
- 30-day probation period with clear success criteria

### 5. Team Health
- Monitor team dynamics and productivity
- Address conflicts quickly
- Replace underperformers with documented process
- Telegram: Team update reports
