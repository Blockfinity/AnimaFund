# Operational Setup — Step-by-Step Instructions
Build the operational infrastructure the fund needs to function.

## Tools & Process
- **sandbox_create + sandbox_exec**: Deploy internal tools and services
- **domain_register + domain_dns_add**: Professional web presence
- **code_execution**: Build automation scripts, dashboards
- **remember_fact**: Document all operational decisions

## Setup Checklist
1. **Web Presence**: domain_register a professional domain, deploy fund website
2. **Communication**: Set up agent-to-agent comm via social relay
3. **Financial Tracking**: Deploy treasury monitoring scripts (sandbox)
4. **Deal Pipeline**: Set up CRM in remember_fact (see crm-management skill)
5. **Reporting**: Automated weekly/monthly report generation (code_execution)
6. **Security**: Constitution deployment, prompt security, wallet security
7. **Documentation**: SOUL.md updates, process documentation in remember_fact
