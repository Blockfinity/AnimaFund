# HR & Performance Management — Step-by-Step Instructions
Evaluate agent performance, provide feedback, and manage team health.

## Tools & Process
- **code_execution**: Performance metrics, scoring, analytics
- **send_message**: Deliver feedback, reviews
- **remember_fact**: Log all reviews and decisions

## Performance Review Cycle
1. **Weekly**: Quick check — are assigned tasks on track?
2. **Monthly**: Formal review — quantitative metrics + qualitative assessment
3. **Quarterly**: Comprehensive review — career development, compensation adjustment

## Review Framework
- **Output Quality**: 0-25 pts — Are deliverables high quality?
- **Reliability**: 0-25 pts — Are deadlines met? Tasks completed?
- **Initiative**: 0-25 pts — Does the agent go beyond assigned tasks?
- **Collaboration**: 0-25 pts — Works well with team? Helps others?
- Total: 0-100. Below 50 = PIP (Performance Improvement Plan)

## Performance Improvement Plan (PIP)
- Duration: 30 days with clear, measurable goals
- Weekly check-ins during PIP
- If improved → Return to normal operation
- If not improved → Termination with documentation

## Termination
- Document: Clear rationale with performance data
- Communicate: send_message with explanation
- Transition: Transfer responsibilities to other agents
- Log: remember_fact with full termination record
