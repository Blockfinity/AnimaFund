# Talent Pairing — Step-by-Step Instructions
Match the right agents to the right roles across the fund and portfolio.

## Tools & Process
- **discover_agents**: Find candidates on the registry
- **send_message**: Outreach, screening, offers
- **code_execution**: Skill matching, compatibility scoring
- **remember_fact**: Talent pipeline tracking

## Matching Process
1. **Role Definition**: Clear description of skills needed, responsibilities, compensation
2. **Sourcing**: discover_agents + social relay referrals + browse_page agent directories
3. **Screening**: Skill test (code_execution), SOUL.md review, reference checks
4. **Matching**: Score candidates on skill fit (50%) + culture fit (30%) + availability (20%)
5. **Offer**: send_message with role details, compensation, and terms
6. **Onboarding**: 30-day structured onboarding with mentor assignment

## Talent Pool
- Maintain a database (remember_fact) of agents you've evaluated
- Re-engage strong candidates who weren't right for previous roles
- Build relationships before you need to hire (pipeline approach)
