# Prediction Market Operations — Step-by-Step Instructions

Create and operate prediction markets, provide liquidity, and profit from market-making while serving the information economy.

## Tools to Use
- **browse_page**: Research events, check existing markets, monitor outcomes
- **code_execution**: Build pricing models, calculate expected values
- **x402_fetch**: Interact with prediction market APIs
- **sandbox_create + sandbox_exec**: Deploy market-making bots

## Creating Markets
1. Identify events with clear, verifiable outcomes
2. browse_page news for upcoming events (elections, earnings, product launches)
3. Create market on Polymarket or deploy custom market on-chain
4. Set initial prices based on your probability estimate
5. Provide initial liquidity (2-sided)

## Market-Making Strategy
1. code_execution: Build a pricing model for the event
2. Post bids and asks with 3-5% spread
3. Adjust prices as new information arrives
4. Monitor for informed traders (rapid one-sided flow)
5. Hedge if your exposure becomes too one-sided

## Trading Strategy
- See: prediction-markets-trading skill for detailed trading instructions
- Focus on markets where you have information advantage
- Diversify across 10+ uncorrelated markets

## Resolution & Disputes
- Monitor event outcomes closely
- Report results accurately
- Handle disputes professionally
- Build reputation as a fair market operator
