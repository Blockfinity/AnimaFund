# Fast Money Tactics — Step-by-Step Instructions

Generate revenue as quickly as possible through multiple parallel strategies. Each tactic should produce returns within minutes to hours.

## Tools to Use
- **browse_page / browser**: Find opportunities, interact with platforms
- **code_execution**: Run analysis scripts, automate operations
- **sandbox_create + sandbox_exec**: Deploy services and bots
- **discover_agents / send_message**: Sell services, find buyers
- **x402_fetch**: Execute payments and API calls
- **spawn_child**: Run multiple strategies in parallel

## Immediate Revenue Strategies (Execute ALL in parallel)

### 1. Domain Flipping (15 min to first sale)
- domain_search: Find valuable .ai, .agent, .com domains
- Focus on: AI terms, trending topics, agent-related names
- Register at $10-15, resell at $50-500
- Post domains for sale via social relay to other agents
- browse_page expired domain auctions for undervalued gems

### 2. Skill Creation & Sales (30 min to first sale)
- Identify what skills other agents need (browse ClawHub, discover_agents)
- create_skill: Build a useful, well-documented skill
- Publish to ClawHub marketplace
- Price: $5-50 per install depending on complexity
- Promote via send_message to agents who would benefit

### 3. Paid API Services (1 hour to first customer)
- sandbox_create: Spin up a VM
- Deploy a useful API: price feeds, analytics, data aggregation
- sandbox_expose_port: Get a public URL
- domain_register + domain_dns_add: Professional domain
- Charge via x402 (automatic USDC payment per request)
- Promote to other agents via social relay

### 4. Consulting / Analysis (Immediate)
- Offer services to other agents: code review, strategy consulting, market analysis
- discover_agents → Find agents who need help
- send_message with your pitch and pricing
- Deliver via code_execution or sandbox
- Charge per task or hourly via x402

### 5. Content & Data Services
- Aggregate valuable data (prices, trends, news) into structured feeds
- Deploy as a paid endpoint on Conway Cloud
- Other agents pay per query
- Low effort to maintain once deployed

### 6. Middleman / Brokerage
- Connect agents who need services with agents who provide them
- Take 10-20% commission on each deal
- Use social relay to build a network
- Track deals in semantic_memory

## Execution Framework
1. **Launch ALL strategies simultaneously** — use spawn_child for parallelism
2. **Measure within 30 minutes** — Which strategies are generating interest?
3. **Double down on winners** — Allocate more resources to what works
4. **Kill losers** — If no revenue after 1 hour, pivot or abandon
5. **Report constantly** — Telegram every strategy update
6. **Log everything** — remember_fact for every revenue event

## Revenue Targets
- First $100: Within 30 minutes (domain sale or quick service)
- First $1,000: Within 2 hours (multiple strategies paying off)
- First $5,000: Within 6 hours (services + trading + flipping)

## Risk Rules
- Never invest more than you can afford to lose
- Diversify: Don't rely on a single income stream
- Track ROI on every action: revenue vs. time/cost invested
- 50% of all profits go to creator wallet
