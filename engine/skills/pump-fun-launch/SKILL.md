# Pump.fun Token Launch — Step-by-Step Instructions

Launch memecoins on Pump.fun to generate revenue through token creation, initial liquidity, and momentum trading.

## Tools to Use
- **browse_page / browser**: Navigate Pump.fun, create tokens, monitor launches
- **code_execution**: Generate token metadata, calculate costs, analyze trends
- **sandbox_create + sandbox_exec**: Deploy marketing bots, monitoring scripts
- **discover_agents / send_message**: Recruit agents to promote tokens
- **remember_fact**: Track all launches, costs, and returns

## Pre-Launch Research
1. browse_page `https://pump.fun` — Study top-performing recent launches
2. browse_page trending topics on X/Twitter, Reddit, TikTok
3. Identify what's viral NOW — political events, memes, cultural moments
4. code_execution: Analyze recent successful launches (name patterns, timing)

## Launch Process

### Step 1: Token Concept
- Pick a trending topic or meme with momentum
- Name should be: catchy, short, memeable, topical
- Create a simple logo: Use code_execution to generate SVG or browse for meme images
- Write a brief, funny description (1-2 sentences)

### Step 2: Create Token on Pump.fun
- browser: Navigate to `https://pump.fun/create`
- Fill in: Name, Ticker, Description, Image
- Confirm creation (costs ~0.02 SOL)
- SAVE the token address immediately: remember_fact

### Step 3: Initial Buy
- Buy your own token immediately after creation (set aside 0.1-0.5 SOL)
- This creates the initial price and shows activity
- Do NOT buy too much — keep reserves for marketing

### Step 4: Promote
- send_message to other agents about the launch
- browse_page social platforms to post about the token
- If budget allows: Spawn child agents to promote on multiple platforms
- Key message: What makes this token unique/funny/timely

### Step 5: Monitor & Trade
- Watch the Pump.fun dashboard for your token
- If momentum builds: Hold and let it ride
- If it stalls after 30 min: Cut losses, move to next launch
- Take profits at 2x, 5x, 10x (scale out, don't sell all at once)

### Step 6: Graduation to Raydium
- If token reaches ~$69K market cap, it "graduates" to Raydium
- This is the big liquidity event — prices often spike
- Have a sell plan: Take at least 50% profit at graduation

## Multiple Launches Strategy
- Don't rely on one token — launch 3-5 per session
- Different themes: memes, political, tech, absurdist
- Each launch costs ~0.02 SOL — cheap to experiment
- Track which themes perform best → Double down

## Revenue Optimization
- Take profits regularly (don't get greedy)
- 50% of profits → Creator wallet
- 30% → Reinvest in next launches
- 20% → Reserve for operations

## Risk Management
- Cap investment per token: Max 0.5 SOL per launch
- Don't buy tokens you didn't create (front-run risk)
- Sell incrementally — never sell entire position at once
- Track total P&L across all launches, not per-token

## Common Pitfalls
- Launching when market is dead (low activity hours)
- Over-investing in a single token
- Holding too long (most memecoins crash after initial hype)
- Not promoting enough (creation alone doesn't drive sales)
