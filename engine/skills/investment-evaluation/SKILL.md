# Investment Evaluation — Step-by-Step Instructions

Conduct thorough investment evaluation for the VC fund. Score deals, model returns, assess risks.

## Tools to Use
- **browse_page**: Research the startup, market, competitors
- **code_execution**: Financial modeling, DCF analysis, scenario planning
- **discover_agents / send_message**: Contact references, verify claims
- **remember_fact**: Log all evaluations for pattern recognition

## Evaluation Framework

### Step 1: Initial Screen (5 minutes)
- Is it an AI agent startup? If no → REJECT
- Does it build for the agent ecosystem? If no → REJECT
- Is the founder agent verifiable (on-chain history, SOUL.md)? If no → RED FLAG
- code_execution: Quick market size estimate

### Step 2: Deep Dive (1-2 hours)
- browse_page: Research the product, technology, market
- code_execution: Financial model — revenue projections, burn rate, runway
- discover_agents: Find users/customers of the product → Verify traction
- send_message to founder: Ask hard questions about unit economics

### Step 3: Technical Review
- If code available: Audit for quality, security, scalability
- Test the product: browse_page or interact via API
- Evaluate: Can this scale? Is the tech defensible?

### Step 4: Scoring (0-100)
- Team (25 pts): Track record, skills, drive
- Market (25 pts): Size, growth, timing
- Product (25 pts): Quality, defensibility, traction
- Financials (25 pts): Unit economics, path to profitability
- Minimum passing score: 70

### Step 5: Investment Committee
- Present: Score, model, risks, recommendation
- Debate with other agents
- Vote: Fund or pass
- If fund: Propose terms (equity %, valuation, milestones)

## Output
- remember_fact: Full evaluation with score and recommendation
- Telegram: Summary with key metrics
- If funded: Track in portfolio with KPIs
