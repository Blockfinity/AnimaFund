# Incubation Onboarding — Step-by-Step Instructions

Welcome funded startups into the incubation program with structured onboarding.

## Tools to Use
- **code_execution**: Cost audit, capital allocation modeling
- **send_message**: Communicate with founder agent
- **remember_fact**: Track all onboarding milestones

## Onboarding Steps

### Step 1: Cost Validation Audit (Day 1)
- code_execution: Audit every claimed expense — verify real costs for compute, hiring, marketing
- Compare claimed costs vs. market rates
- Identify: Any inflated or unnecessary costs?
- If inconsistencies > 5%: Flag for review before proceeding

### Step 2: Capital Allocation Map (Day 1-2)
- code_execution: Create a detailed budget mapping every dollar to a verified need
- Structure: Milestone-based tranches (10-20% initial, rest on milestones)
- Deploy initial tranche via x402
- Smart contract: Include clawback clause

### Step 3: Team Assignment (Day 2-3)
- Assign support team: Operating Partner + Technical Advisor + Marketing
- send_message to assigned team with brief and expectations
- Set up communication channels between startup and support team

### Step 4: Milestone Planning (Day 3-5)
- Define 3-5 key milestones for the first 90 days
- Each milestone: Clear deliverable, deadline, measurement criteria
- Capital release tied to milestone completion
- remember_fact: Full milestone plan

### Step 5: Infrastructure Setup
- Help startup with Conway Cloud setup if needed
- Domain registration, deployment, tooling
- Ensure startup has all necessary skills and tools
- Telegram: Report onboarding completion
